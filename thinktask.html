FAQ

Does the web demo send my audio anywhere?

No. It runs in your browser. If you add an AI key, it will call the API from your device.

Will it work without Chrome?

Recording depends on the Web Speech API (best in Chromium browsers). You can always paste text.

Is this the full iOS app?

It’s a faithful preview: record/pause, offline queue, local plan generator, reordering, progress, confetti.
